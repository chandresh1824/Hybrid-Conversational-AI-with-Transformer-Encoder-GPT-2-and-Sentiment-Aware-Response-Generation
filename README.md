# Hybrid-Conversational-AI-with-Transformer-Encoder-GPT-2-and-Sentiment-Aware-Response-Generation
This project implements a hybrid conversational AI system that combines a fine-tuned GPT-2 with a custom Transformer encoder to generate context-aware, stylistically consistent, and sentiment-adaptive responses from conversational datasets . The model learns from structured dialogue data (PersonaChat-style) and enhances generation using multiple techniques including conceptual embedding injection, answer-vocabulary biasing, sentiment-driven token weighting, and conversational memory tracking for multi-turn coherence. It also integrates fallback generation via BlenderBot for out-of-domain queries and applies post-processing to ensure readability and human-like phrasing. The system evaluates performance using advanced NLP metrics such as BERTScore, SBERT similarity, MAUVE, confidence scoring, and Turing-style classification, while also supporting training visualization, embedding alignment analysis, and A/B comparison between GPT-2, Transformer-only, and hybrid outputs—making it a comprehensive framework for building high-quality, human-like conversational agents.
